\# Investment Portfolio Analyzer with Risk Assessment



A comprehensive portfolio analysis tool designed to automate risk assessment and provide actionable investment recommendations for retail banking clients.



\## Business Context

This system addresses the need for automated portfolio risk assessment in retail banking, reducing manual analysis time while improving accuracy and consistency in investment advisory services.



\## Features

\- Real-time portfolio risk calculation (Sharpe ratio, beta, volatility)

\- Automated diversification analysis

\- Risk tolerance alignment scoring

\- Rebalancing recommendations

\- Compliance-aligned reporting



\## Tech Stack

\- \*\*Python 3.10+\*\*

\- \*\*Pandas, NumPy\*\* - Data processing

\- \*\*yfinance\*\* - Market data API

\- \*\*Streamlit\*\* - Interactive dashboard

\- \*\*Statistical models\*\* - Risk assessment
